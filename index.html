Mince

Lightweight database ORM for Ruby

Download Download View Mince on GitHub

1. Install

$ gem install mince

# Install one of the mince compatible database interfaces
$ gem install hashy_db

2. Use

# Tell Mince what database interface to use
Mince::Config.interface = Mince::HashyDb::Interface

interface = Mince::Config.interface
interface.add 'books', title: 'The World In Photographs', publisher: 'National Geographic'
interface.add 'books', title: 'The Flinch', publisher: 'Julien Smith'
interface.find_all 'books'
